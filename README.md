# Sophia

> _**sophia** / *n.* / (Greek: *σοφία*)_ <sub>so‧phi‧a</sub>
> 1. _a female given name from Ancient Greek_
> 2. _wisdom_

**Sophia** was born as a challenge to create a performant chess engine from scratch in pure Python. It serves as a didactic experiment to question the necessity of compiled languages in chess programming, leveraging bitboards and efficient pruning techniques to deliver speed through the standard UCI protocol.
